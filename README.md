## Features

1. Clock
- 현재 시간을 확인할 수 있다.
2. Username Persistance
- 사용자의 이름을 입력하여 화면에 인사말을 띄울 수 있다.
- 사용자가 다시 페이지를 로드하더라도 저장된 이름을 불러와서 사용할 수 있다.
- 사용자 이름을 클릭하여 기존에 저장된 이름을 지우고 새로운 이름을 설정할 수 있다.
